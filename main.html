대시보드 선택

두 가지 대시보드 중 원하는 화면을 선택하세요.
각각 다른 목적과 기능을 가지고 있습니다.

🍽️

식당 코너별 이용자 현황

식당의 각 코너별 실시간 이용자 수와 통계를 모니터링하는 대시보드

  • 6개 코너별 실시간 이용자 현황
  • 원형 진행률 그래프
  • 사이드바 방문 차트
  • 식사시간별 데이터 전환
  • 테이크아웃 주문 현황
 🤖

로봇 자동화 관제 총괄 화면

5개 사이트의 로봇 자동화 시스템을 총괄 관리하는 관제 대시보드

  • 3개 권역별 사이트 관리
  • 장비 상태 실시간 모니터링
  • 도면기반 모니터링 화면 연동
  • 사이트별 운영 상태 표시
  • 장애/가동 상태 알림
 🏗️

도면 기반 설비 모니터링

실제 도면 위에서 각 설비의 위치와 상태를 직관적으로 모니터링하는 대시보드

  • 실제 도면 기반 설비 위치 표시
  • 설비별 실시간 상태 모니터링
  • 마우스 호버 툴팁 정보
  • 클릭 시 상세 정보 패널
  • 에러/경고 상태 시각적 표시

💡 Tip: 각 대시보드는 독립적으로 작동하며, 브라우저 탭을 여러 개 열어 동시에 모니터링할 수 있습니다.
